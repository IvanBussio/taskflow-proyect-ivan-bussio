Bienvenido a TaskFlow

TaskFlow es una aplicación diseñada para ayudarte a organizar tus tareas diarias de forma clara, simple y visual.

¿Qué puedes hacer?

✔ Crear tareas ✔ Marcar tareas completadas ✔ Buscar y filtrar tareas ✔ Ordenar alfabéticamente ✔ Mantener tu productividad bajo control

Quiénes somos

TaskFlow es un proyecto creado durante un bootcamp de desarrollo web para aprender JavaScript, diseño de interfaces modernas y gestión de datos en aplicaciones reales.
